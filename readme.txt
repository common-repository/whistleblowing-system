== Description ==

"Whistleblowing system" form is the ultimate solution for effortlessly creating and managing contact and whistleblowing forms on your WordPress website.
With its user-friendly interface and powerful features, "Whistleblowing system" empowers website owners to seamlessly connect with their visitors and customers.
It is an advanced whistleblowing protection system that enables secure and anonymous reporting with randomly generated password for personalized login, safeguarding the identity of whistleblowers while ensuring their reports are handled responsibly and confidentially.
The companies can use it to comply with the EU Directive 2019/1937 and manage cases.
This solution follows the EU Whistleblower Directive of 2019 and the GDPR and helps small and large organizations to comply with regulations.

Whistleblowing system suitable for companies and clubs.

= Whistleblowing System Features =

* **Intuitive Interface**: The system has an intuitive interface that helps to complete forms  effortlessly, ensuring a seamless and user-friendly experience.
* **Two-way communication**: This function offers secure, anonymous reporting with two-way communication, GDPR-compliant data privacy, and protection.
* **Confidential or anonymous information**: With the anonymity function the whistleblower doesn't need to  provide an email address. The communication history is documented in the system.
* **Drag & Drop Form**: The drag-and-drop function is an intuitive interface that makes the plugin the most user-friendly form  for WordPress.
* **Free Conditional Logic**: The free Conditional Logic function can be used to show and hide form fields based off of the user's inputs.
* **Form Preview**: This feature enables you to have a form preview without embedding the form anywhere on the website.
* **Mobile-Friendly and Responsive**: With this function your forms will look great on all resolutions and devices: mobile, tablet, and desktop.
* **Field Types**: Plugin offers different form field types that will help you create just the form you need while writing your reports.
* **Personalized email address notification**: With this function it is possible to send a notification to any selected email address when a whistleblower has written a report.

= Advanced Features =

* **Report management within the admin area**: With this function, it is possible to address report tickets straight for the admin area without having to log in with the token.
* **File Upload**: This feature enables file uploads with the option to select specific file types and enforce size limits on the uploaded files.
* **Fully Customizable Themes**: With the variety of themes you can customize the colour scheme to match your own corporate design.
* **Personalized email address notification**: Once your report is submitted you can send notifications to multiple selected email addresses.
* **Export reports as CSV file**: With this function it is possible to export your report as a CSV file.

== Frequently Asked Questions ==

= What is special about this whistleblower protection solution? =
Our solution offers a user-friendly way to implement the Whistleblower Protection Act (HBPG) on your website. It contains functions such as randomly generated passwords, anonymous hints, free Conditional Logic, drag-and-drop form fields and more.

= How can I install the plugin on my WordPress website? =
Installation is simple. Download, install and activate the plugin like any other WordPress plugin.

= Can I customize the contact form? =
Yes, you can customize the form as you wish. Numerous form fields are available. It is also possible to create multiple forms and use them either for whistleblowing or as a regular contact form.
It is also possible to adapt the form to your website design and integrate it on any page using [Shortcodes].

= How does the notification to selected e-mail addresses work? =
You can select any e-mail address to which notifications are automatically sent when someone submits a report.

= Is the plugin optimized for mobile devices? =
Yes, the plugin is 100% responsive and adapts to all screen resolutions and devices.

= Can I protect the anonymity of whistleblowers? =
Yes, whistleblowers have two options:  either remain anonymous or provide their e-mail address. Also, we guarantee that our plugin does not store any personal data, such as IP addresses or any identifiable information.

== Privacy Notices ==
Whistleblowing System plugin does not collect and store any data of your users on Whistleblowing-form's end. All data submitted by your website visitors is stored in your website database. From this perspective, you may be subject to GDPR compliance.

Whistleblowing System imply interaction between website visitors and website owner. As such you may publish forms that require input of Private data. You need to get explicit consent from your users to comply with GDPR. Under GDPR your users may request access  and/or erasure of their entry data at any time. Here you can find how to export and/or delete reports.

== Installation ==

After downloading the ZIP file,

1. Log in to the administrator panel.
2. Go to Plugins Add &gt; New &gt; Upload.
3. Click &quot;Choose file&quot; (&quot;Browse&quot;) and select the downloaded zip file.
 _For Mac Users_
 _Go to your Downloads folder and locate the folder with the plugin. Right-click on the folder and select Compress. Now you have a newly created .zip file which can be installed as described here._
4. Click &quot;Install Now&quot; button.
5. Click &quot;Activate Plugin&quot; button for activating the plugin.

If the installation does not succeed, please [contact us](https://whistleblowing-form.de/en/contact-whistleblowing-system/) for help.
